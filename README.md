Bem-vindo ao repositório de estudos baseado no livro **"Programação em Python 3: Uma introdução completa à linguagem Python"**. Este repositório foi estruturado para servir de diário de bordo prático, organizando anotações teóricas, códigos de exemplo e resoluções de desafios propostos.

---

## 🛠️ Como o Repositório Está Organizado

Cada capítulo possui sua própria pasta dentro do diretório `estudos-python/` e segue uma estrutura padrão e modular para evitar conflitos de execução:

*   **`README.md`**: Anotações teóricas resumidas e explicação dos conceitos e exercícios.
*   **`exemplos/`**: Códigos demonstrados ao longo do capítulo (ex: `exemplos/ola-mundo/hello.py`).
*   **`exercicios/`**: Subpastas isoladas para cada exercício, permitindo testar e rodar resoluções de forma independente.

---

## 🚀 Como Executar os Códigos

Certifique-se de ter o Python 3 instalado e siga as diretrizes abaixo no terminal.

### Rodar diretamente (Modo Desenvolvimento)
Para testar o código rapidamente:
```bash
python estudos-python/capitulo01/exemplos/ola-mundo/hello.py
```
*(Caso esteja em sistemas baseados em Unix, pode ser necessário rodar com `python3`).*
