**Programação em Python 3: Uma introdução completa à linguagem Python**

Python é provavelmente a linguagem de programação popular mais fácil e agradável de lidar. O código Python é simples de ler e escrever e consegue ser conciso sem ser enigmático. Python é uma linguagem muito expressiva, o que significa que podemos normalmente escrever em muito menos linhas de código em comparação ao que seria necessário em uma aplicação equivalente, digamos, em C++ ou Java. 

Python é uma linguagem multiplataforma. Em geral, o Python pode ser executado no Windows e em sistemas derivados do Unix — como Linux, BSD e macOS (OS X, etc.) — por meio da cópia do arquivo ou arquivos que compõem o programa para a máquina alvo, sem nenhuma construção ou compilação necessária. É possível criar programas Python que usam funcionalidades de uma plataforma específica, mas é raro que isso seja realmente necessário, já que quase todas as bibliotecas Python (tanto as padrão quanto as de terceiros) são completa e transparentemente multiplataformas.

Um dos pontos fortes do Python é que ele vem com uma biblioteca padrão bastante abrangente. Isso nos permite fazer coisas como baixar um arquivo da internet, descompactar um pacote de arquivo compactado ou criar um servidor web, tudo isso com uma ou poucas linhas de código. Em adição à biblioteca padrão, milhares de bibliotecas de terceiros estão disponíveis, algumas providenciando mais poder e facilidades sofisticadas. Por exemplo, a biblioteca de rede *Twisted* e a biblioteca numérica *NumPy*, enquanto outras fornecem funcionalidades muito especializadas que não precisam ser incluídas na biblioteca padrão, como o pacote de simulação *SimPy*. 

A grande parte das bibliotecas de terceiros está disponível pelo Índice de Pacotes do Python (PyPI), no endereço pypi.python.org/pypi. O Python pode ser usado para programação procedural, orientada a objetos e, para extensões menores, em estilo funcional, ainda que o seu coração seja de uma linguagem orientada a objetos. Este livro mostra como escrever programas procedurais e orientados a objetos, e ensina as características da programação funcional do Python.

O objetivo deste livro é mostrar como escrever programas em um bom estilo idiomático Python 3, além de ser uma referência proveitosa à linguagem após a leitura inicial. Ainda que o Python 3 seja uma evolução, em vez de um avanço revolucionário do Python 2, algumas práticas antigas não são mais apropriadas ou necessárias, assim como novas práticas foram introduzidas para melhorar seus recursos. Python 3 é uma linguagem melhor do que o Python 2. Foi construída com base em anos de experiência e adiciona muitas novas características, omitindo inconveniências da versão anterior para torná-la mais agradável de usar, bem como muito mais conveniente, fácil e consistente.

O objetivo deste livro é ensinar a linguagem Python e, ainda que muitas das bibliotecas padrão sejam utilizadas, nem todas estão aqui. Isso não será um problema, pois uma vez que você tenha lido o livro, terá adquirido conhecimento suficiente para fazer uso de qualquer biblioteca padrão ou de terceiros, além de estar apto a criar módulos de bibliotecas por si mesmo.

O livro foi elaborado de forma a ser útil para vários tipos de público, incluindo autodidatas, programadores amadores, estudantes, cientistas, engenheiros e outros que precisam de programação como parte do seu trabalho, além, é claro, de profissionais e cientistas da computação. Para ser capaz de atender a um leque tão grande de pessoas sem, no entanto, intimidar os profissionais ou deixar para trás os iniciantes no assunto, o livro assume pelo menos alguma experiência prévia em programação em qualquer linguagem. Em particular, adota um conhecimento básico de tipos de dados (como números e *strings*), coleções (como *sets* e *lists*), estruturas de controle (como comandos *if* e *while*) e funções. Adicionalmente, alguns exemplos e exercícios requerem conhecimento básico de HTML, e alguns dos capítulos mais especializados no final demandam o conhecimento básico de suas respectivas áreas de atuação (por exemplo, o capítulo de banco de dados assume um conhecimento básico de SQL).

O livro foi estruturado de maneira a torná-lo o mais produtivo e rápido possível. Ao final do primeiro capítulo, você estará apto a escrever pequenos, porém úteis, programas em Python. Cada capítulo introduz novos tópicos e, com frequência, tanto amplia como aprofunda a cobertura dos temas abordados nos capítulos anteriores. Isso significa que, se você ler os capítulos em sequência, poderá parar em qualquer ponto e estará apto a escrever programas completos com o que aprendeu até ali. Mais tarde, é claro, poderá retomar a leitura para aprender técnicas mais avançadas e sofisticadas quando estiver preparado. Por essa razão, alguns tópicos são introduzidos em um capítulo e, então, explorados de modo mais profundo em outros posteriores.

Dois problemas chaves costumam aparecer quando se ensina uma nova linguagem de programação:
1. O primeiro é que, algumas vezes, quando é necessário ensinar um conceito particular, este conceito depende de outro que, por sua vez, depende direta ou indiretamente do conceito inicial.
2. O segundo é que, no começo, o leitor pode saber pouco ou absolutamente nada da linguagem em questão.

Logo, é muito difícil compor exemplos ou exercícios que sejam interessantes ou úteis. Neste livro, procuramos resolver ambos os problemas. Primeiramente, assumimos uma prévia experiência em programação; depois, indo ao coração do Python, apresentamos no capítulo 1 oito peças-chave que são autossuficientes para escrever programas decentes. Uma das consequências dessa abordagem é que, nos primeiros capítulos, alguns dos exemplos são um tanto artificiais no estilo, uma vez que usam apenas o que foi explicado até o ponto em que se está. Esse efeito vai diminuindo capítulo após capítulo até o final do capítulo 7, no qual todos os exemplos são escritos de maneira totalmente natural e no estilo idiomático do Python 3. A abordagem deste livro é totalmente prática e você será encorajado a testar os exemplos e exercícios, visando conquistar maior experiência. Quando possível, programas pequenos, mas completos, serão usados como exemplos a fim de providenciar casos de uso real. As soluções estão disponíveis na internet no site www.qtrac.eu/python3book.html e todos os exemplos foram testados com o Python 3.0 no Windows, Linux e macOS X.

**A Estrutura do Livro:**
*   **Capítulo 1:** Apresenta oito peças-chave do Python que são suficientes para escrever programas completos. Também descreve alguns dos ambientes de programação disponíveis e mostra dois pequenos exemplos de programas.
*   **Capítulos 2 ao 5:** Introduzem as características da programação procedural do Python, incluindo tipos de dados, coleções, muitas funções internas úteis e estruturas de controle, assim como a manipulação de arquivos de textos muito simples. O capítulo também mostra como criar módulos e pacotes personalizados, além de fornecer uma visão geral da biblioteca padrão para evitar reinventar a roda.
*   **Capítulo 6:** Lhe dará uma introdução completa à programação orientada a objetos (POO) com Python. Todo o material da programação procedural aprendido anteriormente ainda será aplicado neste passo, uma vez que a POO é construída sobre fundações procedurais.
*   **Capítulo 7:** Cobre a parte de escrita e leitura de arquivos. Para arquivos binários, a cobertura inclui compressão e acesso aleatório; para arquivos de texto, abrange a análise manual e com expressões regulares. Este capítulo também mostra como escrever e ler em arquivos XML usando árvores de elementos DOM (*Document Object Model*) e SAX (*Simple API for XML*).
*   **Capítulo 8:** Revisita grande parte do material já abordado, explorando as mais avançadas características do Python nas áreas de tipos de dados, coleções, estruturas de controle, funções e programação orientada a objetos. Introduz algumas funções novas, classes e técnicas avançadas, incluindo o estilo de programação funcional.
*   **Capítulo 9:** Mostra técnicas para expandir a carga de trabalho de um programa através de processos múltiplos e múltiplas tarefas.
*   **Capítulo 10:** Expõe como escrever aplicações cliente-servidor usando o suporte padrão de rede de comunicação do Python.
*   **Capítulo 11:** Fala sobre a programação de banco de dados, tanto com arquivos de chave-valor simples (DBM) quanto com bancos de dados SQL.
*   **Capítulo 12:** Explica e ilustra as minilinguagens de expressões regulares do Python e demonstra como o módulo funciona.
*   **Capítulo 13:** Introduz a programação de GUI (*Graphical User Interface* / Interface Gráfica do Usuário).

A maioria dos capítulos é um tanto longa para manter todo o material relacionado junto em um lugar que facilite a referência. De qualquer maneira, os capítulos estão repartidos em seções, subseções e, algumas vezes, em sub-subseções. Por isso, é fácil seguir a leitura no ritmo que mais lhe convier.
